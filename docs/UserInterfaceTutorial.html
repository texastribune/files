User Interface

Interactive Example

Below is a file browser with a memory directory as the root directory. By default it is empty. You can add a file to the root directory by dragging and dropping it from your local filesystem. You can also click the gear icon to open the context-menu and add files or directories through that.

After adding a directory, you can double click on that directory to navigate into it.

Columns can be sorted by clicking on the column headers.

The current directory can be searched by typing a search query in the search bar in the upper right.
