Texas Tribune File App

This app powers the file system that the Texas Tribune uses withing its CMS. This app provides a flexible file system api for use in the browser. It strives to have minimal dependencies and use modern javascript to enable better performance and unique features.

Quickstart

Go to /src/index.html for a desktop view using a simple version of a filesystem created using the features in this app.
