Class: Link

Link(fileObject, name, parent)

new Link(fileObject, name, parent)

A FileObject that links to another file object. The file data is a json string with the path of the file it links to.
Parameters:
Name Type Description
fileObject FileObject The file this links to.
name string The name of the link
parent FileObject | null The parent directory for this file. Null if root directory.
