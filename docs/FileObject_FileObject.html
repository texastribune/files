Class: FileObject

FileObject(fileStorage, fileNode, name, parent)

new FileObject(fileStorage, fileNode, name, parent)

An object representing a file or directory in a file system. Provides API for performing operations on the file.
Parameters:
Name Type Description
fileStorage AbstractFileStorage The file system the file is on
fileNode FileNode The FileNode object representing the file
name string The name of this file.
parent FileObject | null The parent directory for this file. Null if root directory.
